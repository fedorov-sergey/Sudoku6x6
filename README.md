# 🧩 Судоку 6×6 с подсказками

Интерактивная игра Судоку на сетке 6×6 с системой подсказок.

## 🌐 Демо

**Ссылка на игру:** [https://fedorov-sergey.github.io/Sudoku6x6/](https://fedorov-sergey.github.io/Sudoku6x6/)

## 🎮 Особенности

- Сетка 6×6 (блоки 2×3)
- Генерация уникальной головоломки при каждом запуске
- 4 типа подсказок: по строке, столбцу, прямоугольнику или конкретной ячейке
- Визуальное выделение подсказки
- Проверка решения
- Клавиатурный ввод (1-6, Backspace)
- Кнопки цифр для ввода мышкой
- Подсветка одинаковых чисел при клике

## 🚀 Как играть

1. Нажмите на пустую ячейку
2. Введите число от 1 до 6 (клавиатурой или кнопками)
3. Используйте подсказки, если застряли
4. Нажмите "Проверить" для проверки

## 🔧 Установка

Просто откройте `index.html` в браузере или используйте GitHub Pages.

## 📁 Структура

- `index.html` — разметка
- `style.css` — стили
- `script.js` — логика игры
- `README.md` — документация

## 📝 Как работают подсказки

- **По строке** — "В 3-й строке заполнено 5 клеток. Найди число в последней ячейке"
- **По столбцу** — "Во 2-м столбце заполнено 5 клеток. Найди число в последней ячейке"
- **По прямоугольнику** — "В левом верхнем прямоугольнике заполнено 5 клеток. Найди число в последней ячейке"
